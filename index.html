About This System

üéØ Project Overview

AgriClimate Q&A is an intelligent information retrieval system that connects users to India's agricultural and climate data through natural language queries. Built as a prototype to demonstrate the power of RAG (Retrieval-Augmented Generation) technology for domain-specific applications.

üîß Technical Stack

Python 3.9+ LangChain RAG OpenAI GPT-4 ChromaDB Flask API Chart.js data.gov.in API IMD Weather API

üîç How RAG Works

  1. Data Ingestion: Agricultural and climate data is fetched from government APIs and stored in a vector database.
  2. Query Processing: User questions are converted into embeddings and semantically matched with relevant data.
  3. Context Retrieval: The most relevant data points are retrieved from the database based on semantic similarity.
  4. Answer Generation: Retrieved data is passed to an LLM which generates natural language responses with proper citations.

üìö Data Sources

  • data.gov.in: Open Government Data Platform of India - providing agricultural production, area, and yield statistics.
  • India Meteorological Department (IMD): Official climate data including rainfall, temperature, and weather patterns.
  • Ministry of Agriculture & Farmers Welfare: Crop-wise and state-wise production data updated annually.
